### Design Requirements
- Audible range (~20Hz-20kHz)
- Sine, Sawtooth, and Square outputs (i hate triangle waves)
- 1V/oct freq control

Not too shabby if I do say so myself. The final thing I need to add is a sine wave output, which should be easy enough.
I've come across a few ways to do this, either just a simple low pass filter, using diodes to sequentially clip the tops of the wave, or using a long tailed transistor pair structure somehow (I need to look into this one more.)

I tried filtering and it works well for one specific frequency but once you start changing it it gets quite gross looking. I'll go with the diode based one as I don't want to deal with transistor matching unless I have to (+ it adds more temperature concerns)

Diode ladder works very well, but it does need specific input range (~5V p2p) and also needs steady +-2.4V voltage references, but I think I might go with this as it's easy to sort out with just a few ldo chips.

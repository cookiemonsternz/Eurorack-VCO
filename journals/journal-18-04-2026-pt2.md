I'd like to have a roughly 12HP module? It's the same as Plaits (very popular vco). I'm not too worried about size, under 15-20HP would be preferable, but if I can get it in 12 that would be sick.

Eurorack uses 3U height, which is 133.35mm. In order to fit between the rails however, the pcb must have a max height of 108-110mm. I've opted for 100mm because jlc pricing :) (over 100mm in any dimension is an extra $6).

For with, each HP is 5.08mm, so 12 is 60.96mm. Going with a 55mm width leaves ~3mm of clearance on each side which I'm happy with.
Heres what it looks like for reference.

![Board size](https://cdn.hackclub.com/019da007-d8d1-7022-b810-216d1cc73161/paste-1776506361192.png)
